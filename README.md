# WeatherApp

A simple weather app that fetches current conditions from the OpenWeather API. Enter a city to see temperature, humidity, wind speed, and a matching icon, with quick Fahrenheit/Celsius toggles.

## Features

- City search with Enter key or button
- Current temperature in °F or °C
- Humidity and wind speed
- Weather condition icons
- Basic invalid city message

## Built With

- HTML
- CSS
- JavaScript
- OpenWeather API

## Getting Started

1. Download or clone this project.
2. Open `index.html` in your browser.

### API Key

This app uses an API key set directly in `index.html`:

- `const apiKey = "YOUR_API_KEY"`

Replace the value with your own key from https://openweathermap.org/api.

## Usage

1. Type a city name in the search field.
2. Press Enter or click the search button.
3. Use `F` or `C` to switch units.

## Assets

Weather icons are stored in `images/`.
